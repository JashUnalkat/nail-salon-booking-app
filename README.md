# 💅 Nail Salon Booking Application

---

## 📖 Project Overview

The Nail Salon Booking Application is a cross-platform mobile application developed using **React Native** and **Expo** that enables customers to conveniently browse nail services, book appointments, and manage their reservations from their mobile devices.

The application integrates **Firebase Authentication** for secure user registration and login, while **Firebase Firestore** is used to store user information, available services, and appointment details. The application provides an intuitive and responsive interface designed to simplify the appointment booking experience for both customers and salon administrators.

Built using modern mobile development practices, the project demonstrates responsive UI development, cloud database integration, authentication, and appointment scheduling.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/JashUnalkat/nail-salon-booking-app.git
```

Navigate to the project folder

```bash
cd nail-salon-booking-app
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npx expo start
```

Open the application using:

- Expo Go
- Android Studio Emulator
- Physical Android Device
